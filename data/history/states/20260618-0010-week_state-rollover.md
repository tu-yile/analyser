# Week State

## 周度运行判断

- 当前周级上下文仅纳入 `2026-06-17` 一个交易日，样本不足，以下判断属于本周初始状态而非完整周结论。
- 本周市场结构暂定为 `偏强但高度分化`，指数层面偏强，赚钱效应层面偏窄。

## 持续主线

- 当前最强主线为 `半导体设备 / 存储芯片 / PCB / 玻璃基板 / 电子化学品`。
- 主线驱动来自 `国内政策支持科技创新 + 交易所制度信号 + 资金主动集中抱团` 的叠加，而不是外盘映射。
- 若后续交易日继续维持量价共振，本周主线应继续围绕科技硬件和科创风格展开。

## 反复风险与失效判断

- 最大风险不是指数转弱，而是 `指数强、个股弱` 的结构继续恶化，导致除主线外的大部分方向持续失血。
- 若科创50失守 `1800`、半导体链只剩高位龙头独涨、成交额明显跌破 `2.8万亿`，则应下调“主线强化”的周度判断。
- 油价回落受益链暂未形成持续主线，后续除非出现显著板块联动，否则不应作为周度核心假设。

## 政策、宏观与产业线索

- `2026 陆家嘴论坛` 的科技金融相关表述，以及上交所、深交所当日披露的制度与产品信号，已成为本周重要政策锚。
- 隔夜美股科技回撤并未主导A股全天定价，说明本周需要把国内政策与产业景气信号放在外盘映射之前。
- 美联储决议仍是周内外部变量，需观察其是否在后续交易日重新约束成长风格。

## 情绪、量能与风格迁移

- 情绪特征是主线强化、非主线承压，属于集中交易而非普遍修复。
- 量能处于高位，`30917.64` 亿元的成交额足以支撑科技主线全天强化，但仍需后续确认是否可持续。
- 风格上，成长显著跑赢价值，科技权重强于普遍小票，红利/周期暂时退居次要位置。

## 后续待验证条件

1. 科创50和创业板指能否在后续交易日延续强势。
2. 半导体和PCB主线能否扩散，而不是只剩核心龙头抱团。
3. 市场广度能否修复，否则本周体验层面的负反馈会积累。
4. 美联储决议后的美元、美债、港股科技走势是否重新压制A股成长风格。
